## 项目概述

RodSki Web Dashboard 是一个基于马里奥游戏主题的 Web 可视化界面，用于展示测试用例、模型、数据和测试结果。

**技术方案**: 纯前端 HTML + CSS + JavaScript，读取 RodSki XML 文件在浏览器端解析，无需后端服务。
