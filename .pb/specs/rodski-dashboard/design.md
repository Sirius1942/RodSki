# RodSki Web Dashboard 设计文档

## 1. 马里奥主题风格定义

### 1.1 配色方案

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| Mario Red | `#E52521` | 主色调、按钮、重点强调 |
| Luigi Blue | `#049CD8` | 辅助色、链接、进度条 |
| Grass Green | `#38B044` | 成功状态、通过标识 |
| Brick Brown | `#C84C0C` | 边框、装饰、悬停状态 |
| Coin Yellow | `#F7D731` | 高亮、金币、星星装饰 |
| Sky Blue | `#5C94FC` | 背景、天空元素 |
| Cloud White | `#F0F0F0` | 卡片背景、文字背景 |
| Dark Brown | `#2C2C2C` | 主文字、深色元素 |

### 1.2 字体规范

```
标题字体：
- Family: 'Fredoka One', cursive
- 用途：页面标题、卡片标题、数字显示
- 备用：'Comic Sans MS', cursive, sans-serif

正文字体：
- Family: 'Nunito', sans-serif  
- 用途：正文内容、表单、说明文字
- 备用：'Segoe UI', sans-serif

像素装饰字体：
- Family: 'Press Start 2P', cursive
- 用途：装饰性文字、游戏化元素
- 备用：无（回退到 Fredoka One）
```

### 1.3 图标与装饰元素

**马里奥经典元素 SVG 图标：**
- :question: 问号砖块 - 功能入口
- :coin: 金币 - 数据指标
- :star: 星星 - 重要标记/评分
- :mushroom: 蘑菇 - 成长/升级指标
- :flower: 花朵 - 特殊功能
- :cloud: 云朵 - 装饰元素
- :hill: 山丘 - 背景装饰

**像素风格边框：**
- 方块化边角（border-radius: 0 或 4px）
- 双线边框效果
- 棋盘格背景图案

## 2. UI 组件设计

### 2.1 卡片组件 (Mario Card)

```
背景：#F0F0F0（浅灰白）
边框：4px solid #C84C0C（棕色）
阴影：4px 4px 0 #2C2C2C（像素阴影）
圆角：8px
内边距：20px

悬停状态：
- transform: translate(-2px, -2px)
- 阴影：6px 6px 0 #2C2C2C
```

### 2.2 按钮组件

**主按钮 (Primary)**
```
背景：#E52521（马里奥红）
文字：#FFFFFF
边框：3px solid #2C2C2C
阴影：3px 3px 0 #2C2C2C
悬停：transform: translate(-2px, -2px)
```

**次按钮 (Secondary)**
```
背景：#049CD8（路易吉蓝）
文字：#FFFFFF
边框：3px solid #2C2C2C
```

**成功按钮 (Success)**
```
背景：#38B044（草地绿）
```

### 2.3 统计卡片

```
尺寸：220px x 160px
布局：图标 + 数字 + 标签
图标：48x48 SVG 马里奥元素
数字：Fredoka One, 48px, #2C2C2C
标签：Nunito, 14px, #666666
```

### 2.4 导航栏

```
高度：70px
背景：#E52521（马里奥红）
图标：左侧 Logo + 问号砖块装饰
菜单项：Nunito Bold, 16px, 白色
悬停：背景加亮 10%
选中：底部 4px #F7D731（金币黄）指示条
```

### 2.5 表格组件

```
表头：#5C94FC（天空蓝）背景，白色文字
行高：56px
斑马纹：奇数行 #FFFFFF，偶数行 #F5F5F5
悬停：#FFF3CD（淡黄色）
边框：2px solid #C84C0C
```

### 2.6 状态标签

| 状态 | 背景色 | 文字色 | 图标 |
|-----|--------|-------|------|
| 通过/Pass | #38B044 | #FFFFFF | :star: |
| 失败/Fail | #E52521 | #FFFFFF | :x: |
| 跳过/Skip | #F7D731 | #2C2C2C | :arrow: |
| 运行中 | #049CD8 | #FFFFFF | :hourglass: |

## 4. 动画与交互

### 4.1 过渡动画

```css
/* 悬停效果 */
transition: all 0.2s ease;

/* 卡片弹起 */
transform: translateY(-4px);

/* 按钮点击 */
transform: translate(2px, 2px);
```

### 4.2 游戏化元素

- 数字增长动画（用例数变化时）
- 金币收集动画（完成测试时）
- 星星闪烁（重要指标）
- 进度条吃豆子动画

### 4.3 加载状态

- 问号砖块旋转加载动画
- 云朵飘动背景

---

## 5. 响应式断点

| 设备 | 宽度 | 布局变化 |
|-----|------|---------|
| 大屏桌面 | >= 1200px | 4列统计卡片 |
| 桌面 | 992px - 1199px | 3列统计卡片 |
| 平板 | 768px - 991px | 2列统计卡片 |
| 移动 | < 768px | 1列，折叠导航 |

---

## 6. 无障碍设计

- 颜色对比度符合 WCAG 2.1 AA 标准
- 所有交互元素可键盘访问
- 使用 aria-label 描述图标按钮
- 焦点状态使用明显轮廓
