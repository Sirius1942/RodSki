# RodSki Web Dashboard 需求文档

## 1. 项目概述

RodSki Web Dashboard 是 RodSki 自动化测试框架的 Web 可视化界面，采用经典马里奥游戏主题风格，为测试团队提供直观、有趣的测试数据管理体验。

### 1.1 核心功能
- 测试用例管理与执行
- 页面对象模型管理
- 测试数据管理
- 测试结果展示与报告

### 1.2 目标用户
- 测试工程师
- QA 团队成员
- 测试项目经理

---

## 2. 功能需求

### 2.1 首页/仪表盘
- **项目统计卡片**
  - 用例总数、今日新增、执行率
  - 模型总数、接口总数
  - 最近一次测试通过率
  
- **最近测试执行结果**
  - 最近 5 次执行记录
  - 执行时间、状态、通过率
  
- **快捷入口**
  - 用例管理
  - 模型管理
  - 数据管理
  - 测试结果
  - 报告中心

### 2.2 用例管理页面
- **用例列表**
  - 支持按名称、标签、状态搜索
  - 按模块、优先级过滤
  - 分页展示（每页 20 条）
  
- **用例详情**
  - 用例基本信息（名称、描述、标签）
  - 测试步骤展示
  - 预期结果
  - 关联的模型和数据
  
- **用例执行历史**
  - 历史执行记录列表
  - 执行状态时间线

### 2.3 模型管理页面
- **模型列表**
  - 模型名称、所属页面、元素数量
  - 支持搜索和过滤
  
- **模型详情**
  - 元素定义列表
  - 定位器信息（ID、 XPath、 CSS）
  - 元素类型
  
- **模型预览**
  - 可视化展示页面结构
  - 元素高亮标记

### 2.4 数据管理页面
- **数据表列表**
  - 数据表名称、记录数、更新时间
  
- **数据表预览**
  - 表格形式展示数据
  - 支持滚动查看
  
- **数据编辑**
  - 行级别编辑
  - 批量导入/导出

### 2.5 测试结果页面
- **历史执行列表**
  - 执行时间、持续时间、状态
  - 通过率统计
  - 支持时间范围过滤
  
- **测试报告详情**
  - 通过/失败用例列表
  - 错误截图展示
  - 执行日志摘要
  
- **通过率统计图表**
  - 饼图：通过/失败/跳过比例
  - 柱状图：趋势变化

### 2.6 报告页面
- **整体报告**
  - 汇总统计数据
  - 关键指标卡片
  
- **详细结果**
  - 按模块分组展示
  - 支持导出 PDF/HTML

---

## 3. 非功能需求

### 3.1 性能需求
- 页面加载时间 < 2 秒
- 数据列表操作响应 < 500ms

### 3.2 兼容性
- 支持 Chrome、Firefox、Safari、Edge 最新版本
- 响应式设计，适配 1280px 以上屏幕

### 3.3 数据源
- 读取 RodSki 项目的 XML 格式文件
- 浏览器端解析，无需后端服务

---

## 4. 技术栈

- **前端框架**：原生 HTML + CSS + JavaScript
- **样式框架**：Tailwind CSS
- **图表库**：ECharts
- **字体**：
  - Fredoka One（圆润可爱标题）
  - Press Start 2P（像素风格装饰）
- **部署**：可集成至 RodSki CLI 启动本地服务
