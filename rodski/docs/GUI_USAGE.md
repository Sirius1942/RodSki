# SKI 测试框架 GUI 使用说明

## 启动界面

```bash
cd rodski
python3 ski_gui.py
```

## 功能说明

### 1. 加载用例
- 点击"加载用例"按钮
- 选择 Excel 测试文件 (.xlsx)
- 左侧树形视图显示所有测试步骤

### 2. 执行测试
- 点击"开始执行"按钮
- 右侧日志窗口实时显示执行过程
- 顶部显示测试结果统计

### 3. 停止测试
- 执行过程中可点击"停止"按钮中断测试

## 界面布局


## 依赖

- PyQt6 >= 6.6.0
- openpyxl >= 3.1.0
- playwright >= 1.40.0
